## Color Scheme

### Radio Buttons
- Selected: Primary color (blue)
- Unselected: Muted gray
- Hover: Light primary color

### File Upload Button
- Background: Secondary color
- Text: Secondary foreground
- Hover: Slightly darker secondary

### Filename Display
- Color: Muted foreground
- Font: Regular weight
- Size: Small (0.875rem)

### Error Messages
- Background: Red/50 (light red)
- Border: Red/200
- Text: Red/800 (dark red)
- Icon: Red/400

## Animation

### Radio Button Selection
- Smooth transition (200ms)
- Fade in/out of input fields
- No jarring layout shifts

### File Upload
- Button hover effect
- Filename fade-in
- Error shake animation (if error)

## Summary

The UI changes are minimal and intuitive:
1. Radio button selection for audio source
2. Conditional rendering of YouTube URL or file upload
3. Clear visual feedback for file selection
4. Consistent error handling and messaging
5. Responsive design for all screen sizes
6. Full accessibility support

The design maintains consistency with the existing UI while adding the new functionality seamlessly.
